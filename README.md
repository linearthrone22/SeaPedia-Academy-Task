# Seapedia Monorepo

Seapedia is a monorepo project containing the backend and frontend components.

## Project Structure

- `backend/` — Node.js Express server with TypeScript, Prisma (PostgreSQL), Zod, and Swagger documentation.
- `frontend/` — Next.js 14 web application with Tailwind CSS, shadcn/ui, and Axios.

## Getting Started

Refer to the README files inside the `backend/` and `frontend/` directories for setup and execution instructions.
